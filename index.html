The Grid

Griglia orizzontale

Il sito è costruito su una griglia di 12 colonne orizzontali.
Gli elementi possono occupare qualunque numero di colonne.

Le colonne sono elementi con classe col-X , con X da 1 a 12.
Le colonne vanno incluse in una riga con classe row.

Per racchiudere un elemento o una riga di colonne entro la larghezza standard di una pagina, racchiudere gli stessi dentro un elemento con classe container

Griglia su display ridotti

La griglia è costruita per essere linearizzata in verticale al di sotto di un singolo breakpoint.

Per default gli elementi .col-X vengono impilati in verticale ed occupano tutta la larghezza del loro elemento padre.

Push e pull

A volte è necessario presentare gli elementi della griglia secondo un ordine orizzontale diverso rispetto al markup (ad. esempio per ragioni di indicizzazione o priorità dei contenuti su display ridotti).

È importante compensare gli elementi che utilizzano le classi push e pull, per evitare sovrapposizioni.

La classe push-X sposta un'elemento a destra di X colonne,
La classe pull-X sposta un'elemento a sinistra di X colonne.

Stile dei blocchi

Larghezza di blocchi

Il contenuto delle pagine del sito è mantenuto all'interno di una larghezza coerente attraverso l'utilizzo della griglia (vedi sezione apposita).

Qualora un elemento debba occupare tutta la larghezza della finestra sarà sufficiente non utilizzare nessuna classe.

Header + Menu

CSS Header

Per avere una header già stilata a livello di CSS crea un tag header ed aggiungi ad esso la classe top-nav-bar.

Per impostare l'altezza della header bisogna modificare il valore della variabile $logoHeight, che puoi trovare all'interno di _measures.scss.

Di base, l'altezza della header sotto i 1023px di larghezza della viewport viene dimezzata. Sei vuoi assegnare un'altezza diversa bisogna cambiare il valore della variabile $logoHeight, che puoi trovare all'interno di _mobile.scss.

Sticky Header

Per ottenere una header sempre posizionata al top della pagina, anche quando essa viene scrollata, si aggiunge ad header la classe sticky-header.

Per evitare che il contenuto HTML che viene dopo la header appaia parzialmente coperto dalla header posizionata fissa al top della pagina, si aggiunge ad essa la classe page-header.

Questa classe CSS fa partire il Javascript che calcola l'altezza dell'header. L'altezza calcolata viene inserita come padding-top del contenuto che viene dopo.
